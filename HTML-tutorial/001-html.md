# HTML 教程

> HTML 是一种网页标准标记语言。你可以使用 HTML 来创建你的网站。HTML 很容易上手，而你也将会喜爱上它的。

## 章节例子

本教程包含了上百个 HTML 的使用例子（demo）。使用我们的在线编辑器，你可以编辑 HTML 代码，通过点击按钮，可以查看编辑结果。

```html
<!DOCTYPE html>
<html>
<head>
<title>Page title</title>
</head>
<body>
<h1>This is a Heading</h1>
<p>This is a paragraph</p>
</body>
</html>
```
